This paper presents LORAL (Lunar Optical Relay And Localization), an embedded infrared (IR) beacon platform that jointly supports optical communication and position estimation using an array of photodiode receivers. Three frequency-separated IR beacons transmit on-off keyed (OOK) signals, enabling frequency-domain channel separation and concurrent packet decoding and signal-strength estimation. A Goertzel-based spectral pipeline is implemented on constrained hardware to extract per-beacon amplitude and identify carrier components for localization.

The system is evaluated over a 0.1--3~m indoor line-of-sight link. Experimental results characterize bit error rate (BER) and packet success rate under varying distance conditions, and demonstrate multi-beacon trilateration achieving sub-10~cm position error in a 3~m~$\times$~3~m workspace. The results show that joint communication and localization can be supported using lightweight spectral processing on embedded platforms without separate sensing modalities.
