# LORAL Experiment Plan

## Research Questions
- What BER and packet success rate can be achieved using BPW34-based IR reception over 0.1–3 m?
- What localization accuracy can be achieved using 3 IR beacons (target: <10 cm, <5°)?

---

## Tasks

1. Develop IR OOK communication protocol and measure BER
2. Characterize BPW34 + MCP6002 response vs distance (0.1–3 m)
3. Demonstrate beacon communication
4. Implement 3-beacon localization in 3m × 3m area
5. Evaluate Q-learning servo optimization for signal acquisition
